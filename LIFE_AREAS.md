## How It Works (Architecture)

- **Memory per Area**: ChromaDB collections or tagged memory so Sierra can instantly recall "everything about my health" or "my career wins since 2023".
- **Life Area Agents**: Specialized sub-agents (or LangGraph nodes) that own one area but talk to the master orchestrator.
- **Proactive Engine**: Daily/weekly briefings, real-time interventions, and "Sierra noticed..." nudges across all areas (voice + push + HUD cards).
- **God Mode Integration**: Once permissions are granted via the big button, Sierra can read/write the actual data sources (Health app, bank APIs, calendars, notes, location) with minimal friction.
- **Unified Command Surface**: "Sierra, give me the state of my life this week" or "How is my health vs finance balance?" or "What should I focus on today for career and relationships?"
